## 🚀 Proyecto Destacado

### Laboratorio Empresarial ASIR

Proyecto integral de administración de sistemas desarrollado como entorno empresarial simulado, diseñado para aplicar conocimientos de infraestructura IT, redes, servicios corporativos y administración de sistemas.

El objetivo del proyecto fue recrear una infraestructura tecnológica similar a la que puede encontrarse en una organización real, documentando su diseño, implementación y gestión.

#### Tecnologías y servicios utilizados

* Windows Server
* Active Directory
* DNS
* DHCP
* Redes TCP/IP
* Virtualización
* Administración de usuarios y permisos
* Compartición de recursos
* Políticas de seguridad
* Microsoft 365
* Documentación técnica

#### Competencias desarrolladas

* Despliegue y administración de servidores
* Gestión de entornos Windows empresariales
* Resolución de incidencias
* Administración de usuarios y grupos
* Configuración de servicios de red
* Implementación de políticas de seguridad
* Documentación y procedimientos técnicos
* Trabajo basado en buenas prácticas de sistemas

#### Resultados

* Infraestructura empresarial funcional.
* Gestión centralizada de usuarios y recursos.
* Servicios de red configurados y documentados.
* Entorno preparado para tareas de soporte y administración.
